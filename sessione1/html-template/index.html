Benvenuto nella mia prima pagina web!

Questa è una pagina HTML di base. HTML (HyperText Markup Language) è il linguaggio standard per creare pagine web.

Cosa puoi fare con HTML:

  • Creare strutture di pagine web
  • Inserire testo, immagini e link
  • Creare form per raccogliere dati
  • E molto altro!

Puoi modificare questo template per creare la tua pagina personale. Prova a cambiare il testo, aggiungere immagini o link!
